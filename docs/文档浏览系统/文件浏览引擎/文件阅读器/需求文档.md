# 文件阅读器 — 需求文档

## 功能概述

在右侧面板展示选中文件的内容，支持 Markdown 渲染和纯文本展示。

## 功能需求

### FR-01: Markdown 渲染
- 调用 `GET /api/file` 获取文件内容
- 使用 marked.js 渲染为 HTML（GFM 模式）
- 支持：表格、任务列表、代码块、引用块、图片

### FR-02: 代码块复制
- 每个 `<pre>` 代码块右上角显示"复制"按钮
- 点击复制代码到剪贴板
- 成功后按钮文字变为"已复制" 2 秒后恢复

### FR-03: 纯文本展示
- `.txt` 文件使用 `<pre>` 标签
- 等宽字体（Consolas/Courier New）
- 保留原始换行和缩进

### FR-04: 文件元信息
- 显示文件名
- 显示文件大小（B/KB/MB）
- 显示最后修改时间

### FR-05: 面包屑导航
- Header 显示当前文件在目录树中的完整路径
- 格式：项目 / 模块 / 功能 / 文件名
