# 文件阅读器 — 开发计划

## 技术选型

| 项 | 选型 | 方案 |
|----|------|------|
| Markdown 渲染 | marked.js v12 | CDN 优先 + 本地降级 |
| 代码高亮 | 无（v1 跳过） | 后续可加 highlight.js |
| 剪贴板 | `navigator.clipboard.writeText()` | 现代浏览器标准 API |

## 开发排期

| 任务 | 工时 |
|------|------|
| `GET /api/file` 服务端端点 | 20min |
| `viewer.js` 文件加载与渲染 | 45min |
| Markdown 渲染 + GFM 配置 | 30min |
| 代码块复制按钮 | 20min |
| 面包屑导航 | 20min |
| 错误处理（文件不存在等） | 15min |

## 关键文件

| 文件 | 职责 |
|------|------|
| `server.js` | `GET /api/file` 端点，路径安全校验 |
| `public/js/viewer.js` | 文件加载、MD 渲染、TXT 展示 |
| `public/index.html` | viewer 面板 DOM 结构 |
