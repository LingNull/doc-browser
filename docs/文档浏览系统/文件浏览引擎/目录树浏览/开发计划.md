# 目录树浏览 — 开发计划

## 技术选型

| 项 | 选型 | 理由 |
|----|------|------|
| 数据格式 | JSON（服务端递归生成） | 前端无需多次请求 |
| 渲染方式 | jQuery DOM 操作 | 轻量，无框架依赖 |
| 图标 | Emoji | 零依赖，跨平台 |
| 搜索 | 前端实时过滤 | 数据量小，无需后端搜索 |

## 开发排期

| 阶段 | 任务 | 工时 |
|------|------|------|
| 服务端 | `GET /api/tree` 递归目录扫描 | 30min |
| 前端 | 树节点递归渲染 `buildNode()` | 1h |
| 前端 | 展开/折叠动画 | 30min |
| 前端 | 搜索过滤 | 30min |
| 前端 | 右键菜单 | 30min |
| 联调 | 与 viewer/generator 联动 | 30min |

## 关键文件

| 文件 | 职责 |
|------|------|
| `server.js` | `GET /api/tree` 端点，`buildTree()` 递归扫描 |
| `public/js/tree.js` | 树渲染、交互、搜索、右键菜单 |
| `public/js/app.js` | 全局状态、树选择事件分发 |
