# 目录树浏览 — 需求文档

## 功能概述

在浏览器左侧面板展示 `docs/` 目录的层级结构，支持展开折叠、搜索过滤、右键操作。

## 功能需求

### FR-01: 递归目录渲染
- 页面加载时调用 `/api/tree` 获取完整目录树 JSON
- jQuery 递归渲染可折叠的树节点
- 默认展开第一级目录

### FR-02: 图标区分
- 📁 有子节点的目录（折叠态）
- 📂 有子节点的目录（展开态）
- 📝 Markdown 文件
- 📄 文本文件

### FR-03: 展开/折叠
- 点击目录节点 → 切换子节点显示/隐藏
- 箭头图标跟随旋转动画
- 点击"全部折叠"按钮收起所有节点

### FR-04: 文件搜索
- 顶部搜索框实时过滤
- 匹配的节点自动展开父级路径
- 不匹配的节点隐藏

### FR-05: 右键菜单
- 右键功能目录 → "生成详细设计"
- 右键任意位置 → "刷新目录"

## 技术实现
- 数据来源：`GET /api/tree` API
- 渲染方式：jQuery DOM 操作，递归 `buildNode()` 函数
- 搜索：`filter()` 遍历所有 `.tree-node-header` 匹配文件名
