# AI 自动归档 — 需求文档

## 功能概述

让 Claude Code 在生成开发文档时，自动将文件存放到 `docs/{项目}/{模块}/{功能}/` 对应的目录中，无需手动搬运。

## 功能需求

### FR-01: AI 读取归档约定
- Claude Code 每次对话自动读取 `CLAUDE.md`
- 其中记录了完整的目录结构和文档命名规范

### FR-02: 智能路径选择
- 生成文档前扫描 `docs/` 下已有的项目、模块
- 如果用户未指定，列出选项让用户选择
- 支持新建项目和模块

### FR-03: 自动创建目录
- 目标路径不存在时自动创建完整的目录链
- 确保目录结构一致

### FR-04: 标准文件命名
- 需求分析 → `需求文档.md`
- 开发计划 → `开发计划.md`
- 代码审查 → `代码审查.md`
- 测试报告 → `测试报告.md`
- 接口文档 → `接口文档.md`
- 数据库设计 → `数据库设计.md`

## 实现机制

| 机制 | 作用 |
|------|------|
| `CLAUDE.md` | 项目级指令，对话自动加载 |
| Memory | 跨会话持久化归档约定 |
| `/save-doc` Skill | 交互式归档命令 |
