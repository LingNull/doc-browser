# AI 自动归档 — 开发计划

## 技术方案

### CLAUDE.md 配置

项目根目录的 `CLAUDE.md` 写入归档约定，包含：
- 三级目录结构说明
- 文档类型与文件名映射表
- 行为规则（询问/扫描/自动创建/告知）

### Memory 持久化

在 `C:\Users\zsh\.claude\projects\D--work-project\memory\` 写入：
- `doc-archiving-convention.md`：归档约定
- `MEMORY.md`：索引

### 自定义 Skill

`/save-doc` 命令的交互流程：
```
Step 1: 扫描 docs/ → 列出已有项目
Step 2: 确认/询问 → 项目、模块、功能、文档类型
Step 3: 创建目录 → New-Item -Force
Step 4: 生成文档 → Write 写入
Step 5: 告知结果 → 路径 + 浏览器地址
```

## 关键文件

| 文件 | 职责 |
|------|------|
| `CLAUDE.md` | 项目指令，AI 每次自动读取 |
| `.claude/skills/save-doc.md` | Skill 定义 |
| 用户 memory 目录 | 跨会话持久化约定 |
