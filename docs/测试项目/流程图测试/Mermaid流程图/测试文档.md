## 1. 自上而下流程图 (flowchart TD)

```mermaid
flowchart TD
    A[用户登录] --> B{验证账号密码}
    B -->|验证通过| C[进入主页]
    B -->|验证失败| D[显示错误提示]
    D --> E{重试次数 < 3?}
    E -->|是| A
    E -->|否| F[锁定账号 30 分钟]
    C --> G[加载用户数据]
    G --> H[渲染仪表盘]
```

## 3. 时序图 (sequenceDiagram)

```mermaid
sequenceDiagram
    participant U as 用户2
    participant F as 前端
    participant S as 后端
    participant DB as 数据库

    U->>F: 点击登录按钮
    F->>S: POST /api/login
    S->>DB: 查询用户信息
    DB-->>S: 返回用户数据
    S->>S: 验证密码哈希
    alt 验证成功
        S-->>F: 返回 JWT Token
        F-->>U: 跳转到主页
    else 验证失败
        S-->>F: 返回 401 Unauthorized
        F-->>U: 显示"密码错误"
    end
```
